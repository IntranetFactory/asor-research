## Question:
what is Workday’s Agent System of Record (ASoR)

## Answer:
Workday’s
**Agent System of Record (ASoR)**
is
a centralized governance and management platform designed to treat AI agents like digital employees
. Just as Workday’s Human Capital Management (HCM) serves as the "source of truth" for human workers, ASoR provides a unified hub to register, monitor, and manage the entire lifecycle of AI agents.
## Key Features of ASoR
- **Agent Lifecycle Management:** It manages agents from "hiring" (registration) to "retirement" (deactivation). Business leaders can assign roles, set permissions, and configure agents similarly to how they manage human staff.
- **Unified Agent Analytics Hub:** A single dashboard that provides real-time visibility into all enterprise agents, whether they are built by Workday, third-party partners (like **Censia**), or the customers themselves.
- **Workday Agent Gateway:** A secure control point that allows both internal and third-party agents to interact with Workday data. It uses open standards like the **Model Context Protocol (MCP)** and **Agent-to-Agent (A2A)** protocols for interoperability.
- **Accountability & Governance:** ASoR enforces strict identity permissions and observability, ensuring that AI agents comply with the same security and data privacy rules as human employees.
## Core Purpose: The "Blended Workforce"
Workday created ASoR to address "agent sprawl"—the chaos caused when different teams deploy unmanaged AI tools across an organization. By bringing agents into the same system as HR and Finance data, ASoR helps companies:
- **Measure ROI:** Track exactly how much value each agent is delivering to the business.
- **Maintain Control:** Monitor which data sources agents are accessing and what actions they are performing.
- **Optimize Productivity:** Understand how humans and AI are working together to identify process efficiencies.
